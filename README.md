## ğŸš€ Getting Started

### Prerequisites

- Node.js v18 or higher
- npm (Node Package Manager) or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd Aawai-website
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```
   Or:
   ```bash
   node server.js
   ```

4. **Open your browser**
   Navigate to: `http://localhost:3000`
